# מדריך העלאה לגיטהאב (GitHub)
## יוחנן פרידמן – ניהול אמנים

---

## שלב 1 – פתיחת חשבון

1. כנס לכתובת: **https://github.com**
2. לחץ על "Sign up" (הרשמה)
3. מלא אימייל, שם משתמש וסיסמה
4. אשר את האימייל שנשלח אליך

---

## שלב 2 – יצירת Repository (ספרייה) חדש

1. לאחר הכניסה, לחץ על **"New"** (כפתור ירוק) בצד שמאל
   - או לחץ על **"+"** בפינה הימנית למעלה ← "New repository"
2. מלא:
   - **Repository name**: `friedman-artists` (שם לאתר שלך)
   - **Description**: ניהול אמנים – יוחנן פרידמן
   - בחר: ● **Public** (ציבורי – חינמי)
   - סמן: ✅ "Add a README file"
3. לחץ **"Create repository"**

---

## שלב 3 – העלאת הקבצים (הדרך הקלה ביותר)

### דרך A – ממשק האינטרנט (ללא התקנות)

1. נכנס ל-Repository שיצרת
2. לחץ על **"Add file"** ← **"Upload files"**
3. גרור את **כל תיקיית הפרויקט** לחלון
   - הקבצים: `index.html`, `admin.html`, `contract.html`, תיקיית `css/`, תיקיית `js/`
4. בשדה "Commit changes" כתוב: `העלאה ראשונית של האתר`
5. לחץ **"Commit changes"** (כפתור ירוק)

⚠️ **חשוב**: בפעם הראשונה תצטרך לגרור קבצים **בנפרד** לכל תיקייה (css ו-js) כי ממשק האינטרנט לא מעלה תיקיות מקוננות בבת אחת.

### דרך B – GitHub Desktop (מומלץ לעתיד)

1. הורד **GitHub Desktop**: https://desktop.github.com
2. הכנס עם חשבון GitHub שלך
3. לחץ "Clone a repository" ← בחר את ה-Repository שיצרת
4. בחר מיקום על המחשב (לא חייב להיות אותה תיקייה)
5. העתק את כל קבצי הפרויקט לאותה תיקייה
6. ב-GitHub Desktop תראה את כל הקבצים החדשים
7. כתוב הודעה (Summary): "עדכון ראשוני"
8. לחץ **"Commit to main"** ← **"Push origin"**

---

## שלב 4 – הפעלת האתר בחינם (GitHub Pages)

זה יאפשר לך לפתוח את האתר מכל מחשב בעולם!

1. נכנס ל-Repository שלך
2. לחץ על **"Settings"** (הגדרות)
3. גלול מטה לסקשן **"Pages"** (בתפריט השמאלי)
4. תחת "Source" בחר:
   - **Branch**: main
   - **Folder**: / (root)
5. לחץ **"Save"**
6. המתן 2-5 דקות
7. תופיע הודעה עם קישור לאתר שלך:
   `https://[USERNAME].github.io/friedman-artists/`

---

## שלב 5 – עדכון האתר בעתיד

כל פעם שאתה רוצה לעדכן קבצים:

**דרך ממשק האינטרנט:**
1. נכנס ל-Repository
2. לחץ על הקובץ שרוצה לעדכן
3. לחץ על עיפרון ✏️ (Edit)
4. ערוך ← לחץ "Commit changes"

**דרך GitHub Desktop:**
1. ערוך את הקבצים במחשב
2. פתח GitHub Desktop – תראה את השינויים
3. כתוב הודעה ← "Commit" ← "Push"
